HTML Styles

The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

Example Code Output
<p>I am normal, There is no any Styling on my Text</p>
<p style="color:red;">My Text Color is RED</p>
<p style="color:blue;">My Text Color is BLUE</p>
<p style="color:darkgreen;">My Text Color is GREEN</p>
<p style="font-size:24px;">My Font Size is Big</p>
<p style="background-color: purple; color: white;">My Background Color is Purple and Text Color is White</p>
<p style="padding-left: 20px; background-color: aquamarine;">My Padding is 20px on Left with a Background Color</p>
<p style="margin-left: 20px; background-color: darkgrey">My Margin is 20px on Left with a Background Color</p>
<p style="text-align:center;">Centered paragraph.</p>

The HTML Style Attribute

Setting the style of an HTML element, can be done with the style attribute.

The HTML style attribute has the following syntax:

Example Code
<tagname style="property:value;">
The property is a CSS property. The value is a CSS value.
