HTML Input Attributes

Let's discuss about the different attributes for the HTML <input> element.

The value Attribute

The input value attribute specifies an initial value for an input field.

Example Code Sample Output
<form>
     <label for="name">Name:</label>
     <input type="text" id="name" name="name" value="Shahlal Hossain">
     <label for="email">Email:</label>
     <input type="text" id="email" name="email" value="shahlal@gmail.com">
</form>

The readonly Attribute

The input readonly attribute specifies that an input field is read-only.
A read-only input field cannot be modified (however, a user can tab to it, highlight it, and copy the text from it).
The value of a read-only input field will be sent when submitting the form.

Example Code Sample Output
<form>
     <label for="name">Name:</label>
     <input type="text" id="name" name="name" value="Shahlal Hossain" readonly>
     <label for="email">Email:</label>
     <input type="text" id="email" name="email" value="shahlal@gmail.com">
</form>

The disabled Attribute

The input disabled attribute specifies that an input field should be disabled.
A disabled input field is unusable and un-clickable.
The value of a disabled input field will not be sent when submitting the form.

Example Code Sample Output
<form>
     <label for="name">Name:</label>
     <input type="text" id="name" name="name" value="Shahlal Hossain" disabled>
     <label for="email">Email:</label>
     <input type="text" id="email" name="email" value="shahlal@gmail.com">
</form>

The size Attribute

The input size attribute specifies the visible width, in characters, of an input field.
The default value for size is 20.
The size attribute works with the following input types: text, search, tel, url, email, and password.

Example Code Sample Output
<form>
     <label for="name">Name:</label>
     <input type="text" id="name" name="name" value="Shahlal Hossain" disabled>
     <label for="pin">PIN:</label>
     <input type="text" id="pin" name="pin" value="" size="4">
</form>

The maxlength Attribute

The input maxlength attribute specifies the maximum number of characters allowed in an input field.
When a maxlength is set, the input field will not accept more than the specified number of characters.
However, this attribute does not provide any feedback. So, if someone want to alert the user, must write JavaScript code.

Example Code Sample Output
<form>
     <label for="sname">Name:</label>
     <input type="text" id="sname" name="name" value="" size="25">
     <label for="spin">PIN:</label>
     <input type="text" id="spin" name="pin" value="" maxlength="4" size="4">
</form>

The min and max Attributes

The input min and max attributes specify the minimum and maximum values for an input field.
The min and max attributes works with the following input types: number, range, date, datetime-local, month, time and week.

Example Code Sample Output
<form>
     <label for="datemax">Enter a Date (Before 2000-01-01:</label>
     <input type="date" id="datemax" name="datemax" max="2000-12-31">
     <label for="datemin">Enter a Date (After 2020-01-01:</label>
     <input type="date" id="datemin" name="datemin" min="2020-01-02">
     <label for="quantity">Quantity (Between 1 and 5):</label>
     <input type="number" id="quantity" name="quantity" min="1" max="5">
</form>


The multiple Attribute

The input multiple attribute specifies that the user is allowed to enter more than one value in an input field.
The multiple attribute works with the following input types: email, and file.

Example Code Sample Output
<form>
     <label for="files">Select Files:</label>
     <input type="file" id="files" name="files" multiple>
</form>

The pattern Attribute

The input pattern attribute specifies a regular expression that the input field's value is checked against, when the form is submitted.
The pattern attribute works with the following input types: text, date, search, url, tel, email, and password.

Example Code Sample Output
<form>
     <label for="country_code">Country Code:</label>
     <input type="text" id="country_code" name="country_code" pattern="[A-Za-z]{3}" title="Three Letter Country Code">
</form>

The placeholder Attribute

The input placeholder attribute specifies a short hint that describes the expected value of an input field
The short hint is displayed in the input field before the user enters a value.
The placeholder attribute works with the following input types: text, search, url, tel, email, and password.

Example Code Sample Output
<form>
     <label for="mobile">Enter Mobile Number:</label>
     <input type="text" id="mobile" name="mobile" placeholder="+880-1717-445566">
</form>

The required Attribute

The input required attribute specifies that an input field must be filled out before submitting the form.
The required attribute works with the following input types: text, search, url, tel, email, password, date pickers, number, checkbox, radio, and file.

Example Code Sample Output
<form>
     <label for="username">Username:</label>
     <input type="text" id="username" name="username" required>
</form>

The step Attribute

The input step attribute specifies the legal number intervals for an input field.
Example: if step="3", legal numbers could be -3, 0, 3, 6, etc.
This attribute can be used together with the max and min attributes to create a range of legal values.
The step attribute works with the following input types: number, range, date, datetime-local, month, time and week.

Example Code Sample Output
<form>
     <label for="points">Points:</label>
     <input type="number" id="points" name="points" min="0" step="3">
</form>

The autofocus Attribute

The input autofocus attribute specifies that an input field should automatically get focus when the page loads.

Example Code Sample Output
<form>
     <label for="customer_name">Name:</label>
     <input type="text" id="customer_name" name="customer_name" autofocus>
     <label for="customer_mobile">Mobile:</label>
     <input type="text" id="customer_mobile" name="customer_mobile">
</form>

The height and width Attributes

The input height and width attributes specify the height and width of an <input type="image"> element.

Example Code Sample Output
<form>
     <label for="user_name">Name:</label>
     <input type="text" id="user_name" name="user_name">
     <label for="user_mobile">Mobile:</label>
     <input type="text" id="user_mobile" name="user_mobile">
     <input type="image" src="images/submit_button.png" alt="Submit" width="120" height="30">
</form>


The list Attribute

The input list attribute refers to a <datalist> element that contains pre-defined options for an <input> element.

Example Code Sample Output
<form>
     <label for="browser">Select Browser</label>
     <input id="browser" list="browsers">
         <datalist id="browsers">
             <option value="Internet Explorer">
             <option value="Mozilla Firefox">
             <option value="Google Chrome">
             <option value="Opera Light">
             <option value="Apple Safari">
             <option value="Microsoft Edge">
         </datalist>
</form>

The autocomplete Attribute

The input autocomplete attribute specifies whether a form or an input field should have autocomplete on or off.
Autocomplete allows the browser to predict the value. When a user starts to type in a field, the browser should display options to fill in the field, based on earlier typed values.
The autocomplete attribute works with <form> and the following <input> types: text, search, url, tel, email, password, datepickers, range, and color.

Example Code Sample Output
<form action="save.php" autocomplete="on">
     <label for="name">Name:</label>
     <input type="text" id="name" name="name">
     <label for="mobile">Mobile:</label>
     <input type="text" id="mobile" name="mobile">
     <label for="email">Email:</label>
     <input type="email" id="email" name="email" autocomplete="off">
     <input type="submit" value="Submit">
</form>
