HTML Links (Hyperlinks)

Links are found in nearly all web pages. Links allow users to click their way from page to page. HTML links are hyperlinks.
One can click on a link and jump to another document. When you move the mouse over a link, the mouse arrow will turn into a little hand.

Note: A link does not have to be text. A link can be an image or any other HTML element!



HTML Links - Syntax

The HTML <a> tag defines a hyperlink. It has the following syntax:

<a href="url">Link Text</a>

The most important attribute of the <a> element is the href attribute, which indicates the link's destination.
The link text is the part that will be visible to the reader. Clicking on the link text, will send the reader to the specified URL address.

Example Code Output
<a href="https://www.google.com/> Visit Google </a>
<a href="https://www.facebook.com/> Visit Facebook </a> 		Visit Google
Visit Facebook

By default, links will appear as follows in all browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red

Tip: Links can of course be styled with CSS, to get different look



HTML Links - The target Attribute

By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.
The target attribute specifies where to open the linked document. The target attribute can have one of the following values:

  • _self - Default. Opens the document in the same window/tab as it was clicked
  • _blank - Opens the document in a new window or tab
  • _parent - Opens the document in the parent frame
  • _top - Opens the document in the full body of the window
Example Code Output
<a href="https://www.google.com/" target="_blank">Visit Google</a> Visit Google (Click to Check)


HTML Links - Absolute URLs vs. Relative URLs

Example Code Output
<h3>Absolute URLs</h3>
<a href="https://www.google.com/">Visit Google</a>
<a href="https://www.facebook.com/">Visit Facebook</a>

<h3>Relative URLs</h3>
<a href="2_basic.html">HTML Basic</a>
<a href="14_images.html">HTML Images</a>

Absolute URLs

Visit Google   |   Visit Facebook

Relative URLs

HTML Basic   |   HTML Images


HTML Links - Image as a Link

To use an image as a link, just put the <img> tag inside the <a>> tag:

Example Code Output
<a href="1_introduction.html">
     <img src="images/html5_introduction.jpg" alt="HTML Tutorial" width="200" height="auto">
</a>


HTML Links - Link to Email Address

Have to use mailto: inside the href attribute to create a link that opens the user's email program.

Example Code Output
<a href="mailto:shahlalhossain@gmail.com"> Send Email </a> Send Email (Click to Check)


HTML Links - Button as a Link

To use an HTML button as a link, One have to add some JavaScript Code. JavaScript allow to specify what happens at certain events, such as a click of a button:

Example Code Output
<button onclick="document.location='1_introduction.html'">HTML Tutorial</button>    (Click to Check)
