HTML Paragraphs

A paragraph always starts on a new line, and is usually a block of text.

The HTML <p> element defines a paragraph. A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

Example Code Output
<p> This is a paragraph. </p>
<p> This is another paragraph. </p>

This is a paragraph.

This is another paragraph.


HTML Display

One cannot be sure how HTML will be displayed.

Large or small screens, and re-sized windows will create different results.

With HTML, one cannot change the display by adding extra spaces or extra lines in your HTML code.

The browser will automatically remove any extra spaces and lines when the page is displayed:

Example Code Output
<p> This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it. </p>

<p> This paragraph
contains       a lot of spaces
in the source         code,
but the     browser
ignores it. </p>

This paragraph contains a lot of lines in the source code, but the browser ignores it.

HTML Horizontal Rules

The <hr> tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.

The <hr> element is used to separate content (or define a change) in an HTML page:

Example Code Output
<h2> What is HTML? </h2>
<p> Details Description of HTML </p>
<hr>
<h2> What is CSS? </h2>
<p> Details Description of CSS </p>
<hr>
<h2> What is JavaScript? </h2>
<p> Details Description of JavaScript </p>
<hr>
<h2> What is PHP? </h2>
<p> Details Description of PHP </p>
<hr>
<h2> What is OOP? </h2>
<p> Details Description of OOP </p>
<hr>
<h2> What is Laravel? </h2>
<p> Details Description of Laravel </p>
<hr>

The <hr> tag is an empty tag, which means that it has no end tag.



HTML Line Breaks

The HTML <br> element defines a line break.

Use <br> if you want a line break (a new line) without starting a new paragraph:

Example Code Output
<p>This is<br>a paragraph<br>with line breaks.</p>

This is
a paragraph
with line breaks.

The <br> tag is an empty tag, which means that it has no end tag.



The HTML <pre> Element

The HTML <pre> element defines preformatted text.

The text inside a <pre> element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks:

Example Code Output
<pre>
    My Bonnie lies over the ocean.

    My Bonnie lies over the sea.

    My Bonnie lies over the ocean.

    Oh, bring back my Bonnie to me.
</pre> 
                                My Bonnie lies over the ocean.

                                My Bonnie lies over the sea.

                                My Bonnie lies over the ocean.

                                Oh, bring back my Bonnie to me.
