HTML Tables

HTML tables allow web developers to arrange data into rows and columns.
A table in HTML consists of table cells inside rows and columns.

Example
SN Name Mobile Email Gender Age City
01 Shahlal Hossain +8801731479874 shahlal@gmail.com Male 25 Rajshahi
02 Kamal Hossain +8801731479875 kamal@gmail.com Male 27 Rangpur
03 Jamal Hossain +8801731479854 jamal@gmail.com Male 35 Sylhet
04 Fatema Jahan +8801731589874 fatema@gmail.com Female 23 Dhaka
05 Habiba Khatun +8801798589874 habiba@gmail.com Female 33 Barisal


Table Cells

Each table cell is defined by a <td> and a <td/> tag.
Everything between <td> and <td/> are the content of the table cell.
td stands for table data.
Note: A table cell can contain all sorts of HTML elements: text, images, lists, links, other tables, etc.



Table Rows

Each table row starts with a <tr> and ends with a <tr/> tag.
You can have as many rows as you like in a table; just make sure that the number of cells are the same in each row.
tr stands for table row.
Note: There are times when a row can have less or more cells than another.



HTML Table - Colspan

To make a cell span over multiple columns, use the colspan attribute:

Example
Name Age
Shahlal Hossain 43
Kamal Hossain 57
Note: The value of the colspan attribute represents the number of columns to span.


HTML Table - rowspan

To make a cell span over multiple rows, use the rowspan attribute:

Example
Name Shahlal Hossain
Mobile +8801731479874
+8801918522392
Note: The value of the rowspan attribute represents the number of rows to span.
