HTML Lists

HTML lists allow web developers to group a set of related items in lists. There are several types of list are available in HTML.

Ordered Lists

  1. Numbers
  2. Uppercase Letters
  3. Lowercase Letters
  4. Uppercase Roman Numbers
  5. Lowercase Roman Numbers
  6. Nested Ordered Lists

Unordered Lists

  • Disc
  • Circle
  • Square
  • None
  • Nested Unordered Lists

HTML Ordered Lists

The HTML <ol< tag defines an ordered list. An ordered list can be numerical or alphabetical.
An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.
The list items will be marked with numbers by default:

Example Code Output
<ol>
     <li>Coffee</li>
     <li>Tea</li>
     <li>Milk</li>
</ol>
  1. Coffee
  2. Tea
  3. Milk


Ordered HTML List - The Type Attribute

The type attribute of the <ol> tag, defines the type of the list item marker. It can have one of the following values:

Type Description
type="1" The list items will be numbered with numbers (default)
type="A" The list items will be numbered with uppercase letters
type="a" The list items will be numbered with lowercase letters
type="I" The list items will be numbered with uppercase roman numbers
type="i" The list items will be numbered with lowercase roman numbers

HTML Unordered Lists

The HTML <ul> tag defines an unordered (bulleted) list.
An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.
The list items will be marked with bullets (small black circles) by default:

Example Code Output
<ul>
     <li>Coffee</li>
     <li>Tea</li>
     <li>Milk</li>
</ul>
  • Coffee
  • Tea
  • Milk


Unordered HTML List - The Type Attribute

The type attribute of the <ul> tag, defines the type of the list item marker. It can have one of the following values:

Type Description
type="disc" Sets the list item marker to a bullet (default)
type="circle" Sets the list item marker to a circle
type="square" Sets the list item marker to a square
type="none" The list items will not be marked


Type wise Unordered List Examples

Example Code Sample Output

<ul type="disc">
     <li>Coffee</li>
     <li>Tea</li>
     <li>Milk</li>
</ul>

HTML Description Lists

HTML also supports description lists.
A description list is a list of terms, with a description of each term. The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term:

Example Code Output
<dl>
      <dt>HTML</dt>
           <dd>HTML is a Markup Language</dd>
      <dt>JavaScript</dt>
           <dd>JavaScript is Scripting Language</dd>
      <dt>PHP</dt>
           <dd>PHP is a Programming Language</dd>
      <dt>SQL</dt>
           <dd>SQL is a Data Query Language</dd>
</dl>
