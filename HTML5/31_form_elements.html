HTML Form Elements

An HTML form is a section of a document which contains controls such as text fields, password fields, checkboxes, radio buttons, submit button etc.
An HTML form facilitates the user to enter data that is to be sent to the server for processing such as name, email address, password, phone number etc.
The HTML <form> element can contain one or more of the following form elements:

  • <label> - It defines a label for an input element.
  • <input> - It defines an input control.
  • <select> - It defines a drop-down list.
  • <option> - It defines an option in a drop-down list.
  • <optgroup> - It defines a group of related options in a drop-down list.
  • <textarea> - It defines a multi-line input control.
  • <button> - It defines a clickable button.
  • <fieldset> - It groups the related element in a form.
  • <legend> - It defines a caption for a <fieldset> element.
  • <datalist> - It specifies a list of pre-defined options for input control.
  • <output> - It defines the result of a calculation.
  • <keygen> - It defines a key-pair generator field for forms.


The <label> Element

The <label> element defines a label for several form elements.
The <label> element is useful for screen-reader users, because the screen-reader will read out loud the label when the user focus on the input element.
The <label> element also help users who have difficulty clicking on very small regions (such as radio buttons or checkboxes) - because when the user clicks the text within the <label> element, it toggles the radio button/checkbox.
The for attribute of the <label> tag should be equal to the id attribute of the <input> element to bind them together.

Example of <label> Element

Example Code Sample Output
<label for="name">Name:</label>
<input type="text" id="name" name="name">


The <input> Element

One of the most used form element is the <input> element.
The <input> element can be displayed in several ways, depending on the type attribute.

Example of <input> Element

Example Code Sample Output
<label for="mobile">Mobile:</label>
<input type="text" id="mobile" name="mobile">


The <select> and <option> Elements

<select> The <select> element defines a drop-down list:
The <option> elements defines an option that can be selected.
By default, the first item in the drop-down list is selected.

Example of <select> and <option> Elements

Example Code Sample Output
<label for="cars">Choose A Car:</label>
<select id="cars" name="cars">
     <option value="volvo">Volvo</option>
     <option value="saab">Saab</option>
     <option value="fiat">Fiat</option>
     <option value="audi">Audi</option>
</select>
To define a pre-selected option, add the selected attribute to the option.
<label for="cars">Choose A Car:</label>
<select id="cars" name="cars">
     <option value="volvo">Volvo</option>
     <option value="saab">Saab</option>
     <option value="fiat" selected>Fiat</option>
     <option value="audi">Audi</option>
</select>
Use the size attribute to specify the number of visible values.
<label for="cars">Choose A Car:</label>
<select id="cars" name="cars" size="3">
     <option value="volvo">Volvo</option>
     <option value="saab">Saab</option>
     <option value="fiat" selected>Fiat</option>
     <option value="audi">Audi</option>
</select>
Use the multiple attribute to allow the user to select more than one value.
<label for="cars">Choose A Car:</label>
<select id="cars" name="cars" size="4" multiple>
     <option value="volvo">Volvo</option>
     <option value="saab">Saab</option>
     <option value="fiat" selected>Fiat</option>
     <option value="audi">Audi</option>
</select>


The <textarea> Element

The <textarea> element defines a multi-line input field (a text area).
The rows attribute specifies the visible number of lines in a text area.
The cols attribute specifies the visible width of a text area.

Example of <textarea> Element

Example Code Sample Output
<label for="message">Message:</label>
<textarea id="message" name="message" rows="5" cols="80"></textarea>

The <button> element defines a clickable button.

Example of <button> Element

Example Code Sample Output
<button type="button" onclick="alert('Hello World')">Click Me</button>


The <button> Element

The <button> element defines a clickable button.

Example of <button> Element

Example Code Sample Output
<button type="button" onclick="alert('Hello World')">Click Me</button>

Note: Always specify the type attribute for the button element. Different browsers may use different default types for the button element.



The <fieldset> and <legend> Elements

The <fieldset> element is used to group related data in a form.
The <legend> element defines a caption for the <fieldset> element.

Example of <fieldset> and <legend> Elements

Example Code Sample Output
<fieldset>
     <legend>Personal Information</legend>
     <form action="save.php" method="POST">
         <label for="username">Name:</label>
         <input type="text" id="username" name="username" value="" placeholder="Enter Your Name">
         <label for="usermobile">Mobile:</label>
         <input type="text" id="usermobile" name="usermobile" value="" placeholder="Enter Your Mobile">
         <label for="useremail">Email:</label>
         <input type="text" id="useremail" name="useremail" value="" placeholder="Enter Your Email">
         <input type="submit" value="Submit" style="font-size: 16px; padding: 5px; float: right">
     </form>
</fieldset>

The <datalist> Element

The <datalist> element specifies a list of pre-defined options for an <input> element.
Users will see a drop-down list of the pre-defined options as they input data.
The list attribute of the <input> element, must refer to the id attribute of the <datalist> element.

Example of <button> Element

Example Code Sample Output
<label for="browser">Select Favorite Browser</label>
<input list="browsers" id="browser">
<datalist id="browsers">
     <option value="Internet Explorer">
     <option value="Firefox">
     <option value="Chrome">
     <option value="Opera">
     <option value="Safari">
</datalist>
