HTML Styles - CSS

CSS stands for Cascading Style Sheets. CSS saves a lot of work. It can control the layout of multiple web pages all at once.

What is CSS?

Cascading Style Sheets (CSS) is used to format the layout of a webpage. With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!


Tip: The word cascading means that a style applied to a parent element will also apply to all children elements within the parent. So, if you set the color of the body text to "blue", all headings, paragraphs, and other text elements within the body will also get the same color!

Using CSS

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements
  • Internal - by using a <style> element in the <head> section
  • External - by using a <link> element to link to an external CSS file

Note: The most common way to add CSS, is to keep the styles in external CSS files.

Inline CSS

An inline CSS is used to apply a unique style to a single HTML element. An inline CSS uses the style attribute of an HTML element.

Example Code Output
<p style="color: red;">
      Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise
</p>
<article style="background-color: darkgreen; color: white;">
      Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise
</article>
<div style="border: 3px solid darkblue;">
      Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise
</div>

Internal CSS

An internal CSS is used to define a style for a single HTML page. An internal CSS is defined in the head section of an HTML page, within a style element.

Example Code Output
<!DOCTYPE html>
<html>
<head>
      <style>
            p {color: red;}
            article {background-color: darkgreen;}
            div {border: 3px solid darkblue;}
      </style>
</head>
<body>
      <p>Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise</p>
      <article>Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise</article>
      <div>Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise</div>
</body>
</html>

Allah will build for him a house in Paradise

External CSS

An external style sheet is used to define the style for many HTML pages. To use an external style sheet, add a link to it in the <head> section of each HTML page:

Example Code Output
<!DOCTYPE html>
<html>
<head>
      <link rel="stylesheet" href="styles.css">
</head>
<body>
      <p>Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise</p>
      <article>Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise</article>
      <div>Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise</div>
</body>
</html>

Whoever builds a mosque for the sake of Allah,
Allah will build for him a house in Paradise


Whoever builds a mosque for the sake of Allah,
Allah will build for him a house in Paradise

Whoever builds a mosque for the sake of Allah,
Allah will build for him a house in Paradise

The external style sheet can be written in any text editor. The file must not contain any HTML code, and must be saved with a .css extension.
Here is what the "styles.css" file looks like:

styles.css
p {color: red;}
article {background-color: darkgreen;}
div {border: 3px solid darkblue;}


CSS Border

The CSS border property defines a border around an HTML element.

Example Code Output
p { border: 3px solid darkorchid; }

Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise



CSS Padding

The CSS padding property defines a padding (space) between the text and the border.

Example Code Output
p { border: 3px solid darkorchid; padding: 30px; }

Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise



CSS Margin

The CSS margin property defines a margin (space) outside the border.

Example Code Output
p { border: 3px solid darkorchid; margin: 30px; }

Whoever builds a mosque for the sake of Allah, Allah will build for him a house in Paradise



Link to External CSS

External style sheets can be referenced with a full URL or with a path relative to the current web page.

Example Code Remarks
<link rel="stylesheet" href="styles.css"> This example links to a style sheet located in the same folder as the current page.
<link rel="stylesheet" href="/css/styles.css"> This example links to a style sheet located in the css folder on the current web site.
<link rel="stylesheet" href="https://www.w3schools.com/html/styles.css"> This example uses a full URL to link to a style sheet.
