HTML Links - Bookmarks

HTML links can be used to create bookmarks, so that readers can jump to specific parts of a web page.
Bookmarks can be useful if a web page is very long.
To create a bookmark - first create the bookmark, then add a link to it.
When the link is clicked, the page will scroll down or up to the location with the bookmark.

Example Code Instruction
<h3 id="Chapter4">Chapter 4</h3> First, use the id attribute to create a bookmark
<a href="#Chapter4">Go to Chapter 4</a> Then, add a link to the bookmark ("Go to Chapter 4"), from within the same page

<a href="#Chapter10">Chapter 10</a>
<a href="#Chapter15">Chapter 15</a>
