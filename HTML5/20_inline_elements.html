HTML Inline Elements

An inline element does not start on a new line.
An inline element only takes up as much width as necessary.
This is a <span> element inside a paragraph

Example Code Output
<a href="#">Link One>/a>
<a href="#">Link Two>/a>
<a href="#">Link Three>/a>
<a href="#">Link Four>/a>
<a href="#">Link Five>/a> 		Link One | Link Two | Link Three | Link Four | Link Five

Note: Though the anchor elements are in different line but they rendered in browser in a same line



Here are the block-level elements in HTML:

<a> <abbr> <acronym> <b> <bdo> <big> <br> <button> <cite> <code> <dfn>
<em> <i> <img> <input> <kbd> <label> <map> <object> <output> <q> <samp>
<script> <select> <small> <span> <strong> <sub> <sup> <textarea> <time> <tt> <var>

Note: An inline element cannot contain a block-level element



The <span> Element

The <span> element is an inline container used to mark up a part of a text, or a part of a document.
The <span> element has no required attributes, but style, class and id are common.
When used together with CSS, the <span> element can be used to style parts of the text:

Example Code Output
<p> National Flag of Bangladesh is containing a <span style="background-color: green; color: white; padding: 10px 15px;">Dark Green</span> Field (in Background) Incorporating with a off-centered <span style="background-color: red; color: white; padding: 10px; border-radius: 50%">Red</span> Disk. </p>
